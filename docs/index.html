Comment retrouver mon code ?

Votre code anonymat = 1ère lettre du prénom + date de naissance (JJMMAA) + 1ère lettre du nom, en majuscules.
Exemple : Jean DUPONT né le 15/03/1998 → J150398D

Mes déclarations d'heures

Rattrapage = heures ajoutées · Retard = déduites · Sortie de stage = temps de formation, compté ou non selon la case. Le planning ci-dessous est établi par le service et ne se modifie pas ici.

Planning établi par le service

Les heures comptabilisées apparaissent sous chaque code. Jour férié : il vous est accordé et déduit de vos heures à réaliser (il ne crée jamais de dette). S'il est travaillé, il ouvre droit à un jour de récupération, posé ensuite sur le planning avec le code RF.
