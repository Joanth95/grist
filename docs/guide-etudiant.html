1. Se connecter

Vous vous connectez avec votre code anonymat. Il se compose de :

1re lettre du prénom + date de naissance (JJMMAA) + 1re lettre du nom, en majuscules.

Exemple : Jean DUPONT, né le 15/03/1998 → J150398D

Ce code est aussi votre identifiant : notez-le, il vous sera redemandé à chaque connexion.

Vous pouvez aussi recevoir un lien de connexion directe intégrant déjà votre code : un simple clic vous connecte, sans avoir à le ressaisir.

2. Enregistrer son entrée en stage

À votre arrivée, si vous n'êtes pas encore enregistré, cliquez sur « Enregistrez votre entrée en stage » sur l'écran de connexion.

Ce qu'on vous demande

  • Votre identité (civilité, nom, prénom, date de naissance, formation, centre de formation, mail, téléphone) ;
  • Votre stage : choisissez d'abord le site (ex. Hôpital de Mercy, Hôpital de Hayange…), ce qui filtre la liste des services proposés ensuite ; puis votre niveau, les dates de début et de fin, et le tuteur si vous le connaissez.
Le tri par site évite les confusions entre services portant le même nom sur des sites différents.

Après validation

  • Votre code anonymat s'affiche — notez-le précieusement ;
  • un message vous invite à confirmer votre inscription auprès du cadre du service (téléphone ou mail affichés).
Important : votre inscription en ligne ne remplace pas la prise de contact avec le cadre. Appelez-le ou écrivez-lui pour confirmer votre arrivée.
Ce formulaire ne sert qu'à votre premier stage (création de votre dossier). Pour un stage suivant, vous avez déjà un code : voir section 7.

3. Lire mon planning

La carte du haut résume votre stage : service, dates, et vos compteurs d'heures.

CompteurSignification
effectuéesHeures déjà réalisées (planning + déclarations validées).
à réaliserObjectif du stage, calculé sur la base de 35 h par semaine.
SoldeDifférence : positif = en avance, négatif = il reste des heures à faire.

En dessous, le planning établi par le service : une carte par semaine. Chaque jour affiche un code horaire et les heures comptabilisées.

Le planning est établi par le service : vous ne pouvez pas le modifier. Les heures sont affichées au format 7h30 (7 heures 30 minutes).

4. Comprendre les jours fériés

Un jour férié est signalé par une pastille férié sur la case du planning.

  • Férié non travaillé : il vous est accordé et déduit de vos heures à réaliser. Il ne crée donc aucune dette — vous n'avez pas à le rattraper.
  • Férié travaillé : vous ouvrez droit à un jour de récupération, affiché sur votre fiche de stage. Le service le pose ensuite sur votre planning avec le code RF, ce qui le décompte.
En résumé : un jour férié ne vous pénalise jamais ; si vous le travaillez, vous gagnez une récupération. (Conforme à l'arrêté du 31 juillet 2009.)

5. Déclarer des heures

Le bouton « + Déclarer » permet de signaler un écart par rapport au planning. Choisissez le type :

TypeEffet sur le compteur
RattrapageHeures faites en plus (heures supplémentaires) → ajoutées.
RetardArrivée en retard → heures déduites.
Sortie de stageTemps de formation (IFSI, AFGSU, regroupement…). Cochez « compte en temps de stage » si ces heures doivent être comptées.

Renseignez la date, l'heure de début et de fin, et un motif ou commentaire si besoin.

Validation par le cadre

Chaque déclaration passe par le cadre :

  • En attente : enregistrée, mais pas encore prise en compte dans le total. Vous pouvez encore la supprimer (🗑️).
  • Validé : validée par le cadre, comptée dans vos heures. Elle ne peut plus être supprimée — contactez le cadre pour toute correction.

6. Si vous avez plusieurs stages

Quand plusieurs périodes vous sont affectées, des onglets apparaissent en haut. Cliquez sur un onglet pour basculer : la carte, les déclarations et le planning s'adaptent à la période choisie. La période en cours est signalée.

7. Ajouter un stage

Vous changez de service ou vous enchaînez sur un nouveau stage ? Depuis votre espace, cliquez sur « + Nouvelle période de stage », en haut de la page. Contrairement au formulaire d'entrée en stage (section 2), vous êtes déjà connecté : il suffit d'indiquer :

  • le site, ce qui filtre la liste des services proposés ;
  • votre niveau, préempli avec le dernier connu (modifiable si vous êtes passé à l'année supérieure, ex. L1 → L2) ;
  • les dates de début et de fin.

Un nouvel onglet apparaît alors dans votre planning (voir section 6), avec le planning à établir par le nouveau service.

Ce formulaire sert uniquement à enregistrer un nouveau stage. Pour un rattrapage, un retard ou une sortie de stage pendant un stage en cours, utilisez plutôt le bouton « + Déclarer » (section 5).

8. Questions fréquentes

J'ai oublié mon code

Recomposez-le : 1re lettre du prénom + date de naissance JJMMAA + 1re lettre du nom. En cas de doute, contactez le cadre de votre service.

Mes heures ne bougent pas après une déclaration

C'est normal : une déclaration n'est comptée qu'une fois validée par le cadre.

Le planning est vide

Il n'a pas encore été rempli par le service. Rapprochez-vous de votre cadre.
